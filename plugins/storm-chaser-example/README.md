# Storm Chaser Example

Example vertical plugin demonstrating how site-specific features attach to JessCMS core.

## Intended for

Storm chaser deployments that need weather alerts, radar embeds, GPS overlays, and livestream pages.

## Provides

- `alert_banner` block
- `plugin_block` with `radar_embed` type
- Admin pages for alerts and radar config
- API routes for alert feeds (future)

## Status

Phase 2 — manifest only. Disabled by default. Enable per deployment when building a storm chaser site.

## Not included yet

- Weather API integrations
- GPS tracking
- Livestream embeds

These will be separate plugins or extensions of this example.
